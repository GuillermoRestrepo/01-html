ETIQUETAS DE SECCIÓN DE CONTENIDO

Sirven para estructurar el contenido de una página web.

(header)

Se utiliza para representar el encabezado de una página web o de una sección de esta. Puede contener títulos, logotipos, menús de navegación, entre otros.

(main)

Se utiliza para representar el contenido principal que se intenta compartir por medio de la página web, solo puede haber uno por documento.

(Footer)

Se usa para representar el pie de página de una página web o de una seccion de esta. Puede contener información de copyright, enlaces a políticas y términos, datos de contacto, iconos de redes sociales, entre otros.

(section)

Sirve para agrupar información relacionada. Ejemplo:

Si es una página que ofrece servicios de desarrollo de software, algunas secciones importantes serían:

· Acerca de...

· Servicios.

· Contacto.

Nota: Cada seccion debe de tener un título descriptivo que indique claramente su propósito, y este debe estar contenido dentro de la etiqueta de encabezado que le corresponda según su jerarquía.

(article)

Sirve para representar contenido independiente, es decir, puede ser entendido sin necesidad de acudir a otras partes de la página web. Algunos ejemplos son:

· Artículos de blogs o noticias.

· Foros o comentarios.

· Reseñas de productos, entre otros.

Nota: Cada article puede tener su propio header y footer de forma opcional. Además, debe contener un título acorde acorde a su jerarquia.

(aside)

Sirve para representar contenido relacionado con el contenido principal, pero que no es indispensable. Ejemplo:

· Barras laterales con información adicional.

· Enlaces relacionados o publicidad.

· Citas destacadas o notas al margen.
