TIPOS DE ELEMENTOS

En HTML, los elementos se dividen en elementos de bloque y elementos en línea según su comportamiento en la estructura de la página.

Elementos De Bloque

Los elementos de bloque ocupan todo el ancho disponible y siempre comienzan en una nueva línea. Se usan principalmente para estructurar el contenido.

Carácteristicas

  • Se extienden a lo ancho de su contenedor.
  • Empiezan en una nueva línea.
  • Pueden contener otros elementos de bloque o en línea.

Ejemplos De Elementos De Bloque

<div>
Agrupa contenido y permite aplicar estilos.
<blockquote>
Representa citas textuales en bloque. Tiene el atributo cite, que sirve para indicar el enlace de la cita.
<pre>
Escribe el texto respetando espacios y saltos de línea. Se utiliza a menudo con la etiqueta <code> para representar código preformateado.
<address>
Representa información de contacto, como una dirección física, correo electrónico, número de teléfono o enlaces a redes sociales. Se suele utilizar con la etiqueta <br> para generar saltos de línea.
<hr>
Genera una línea horizontal que indica un cambio de temática.
<nav>
Sirve para encapsular el contenido de un menú de navegación. Los elementos de los menús de navegación suelen ir listados en una lista desordenada (ul).

Elementos de Línea

Los elementos en línea ocupan solo el ancho necesario para su contenido y no fuerzan un salto de línea. Se usan principalmente para dar formato dentro de un bloque.

Características

  • Solo ocupan el espacio necesario.
  • No empiezan en una nueva línea.
  • Solo pueden contener otros elementos en línea (NO elementos de bloque).

Ejemplos De Elementos En Línea

<small>
Indica que el texto es de menor importancia que el texto normal.
<em>
Indica énfasis en el texto, dándole mayor importancia.
<strong>
Indica que el texto es de mayor importancia que el texto normal y que el texto dentro de una etiqueta <em>.
<i>
Muestra el texto en cursiva.
<b>
Muestra el texto en negrita.
<u>
Muestra el texto subrayado.
<br>
Genera un salto de línea.
<wbr>
Genera un salto de línea en caso de ser necesario.
<sup>
Superíndice (Ejemplo: 32=9).
<sub>
Subíndice (Ejemplo: H2O).
<time>
Representa fecha y hora.
<span>
Ayuda a aplicar estilos en línea.
<q>
Representa citas textuales cortas. Tiene el atributo cite, que sirve para indicar el enlace de la cita.
<cite>
Representa títulos de obras, películas, libros, estudios, entre otros.
<code>
Representa fragmentos de código.
<kbd>
Representa entrada de teclado. Ejemplos:
  • Escriba si si está de acuerdo.
  • Para copiar, utilice Ctrl+C.
  • Para listar el contenido de un directorio, escriba el comando ls.
