ENLACES

Los enlaces, popularmente conocidos por su nombre en inglés (Links)

Tipos De Rutas

Existen dos tipos de rutas.

1. Rutas Relativas

Son todas las rutas que se utilizan para desplazarce entre los archivos del proyecto.

href="/"

Es una ruta relativa a la raíz del proyecto, la cual permite acceder a los archivos q se encuentren en el directorio de mayor jerearquia (el directorio q contiene todo el proyecto) en caso de solo estar escrito el "/", se dirigirá al archivo index.html, es por eso q es aconsejable q el arcihvo principal de cada proyecto sea index.html.

href="archivo"

Para moverse a archivos del mismo directorio. Ejemplo:

Ir A Enlace De Prueba I

href="capeta/archivo"

Para moverse a archivos que se encuentre en directorios dentro del directorio en el q nos encontramos.

Ir A Enlace De Prueba II

href="../archivo"

Para moverse a archivos de directorios superiores "retroceder de directorio".

Ir A Index

2. Rutas Absolutas

Utilizan un protocolo (http:// o https://), y es un enlance externo. Ejemplo:

Ir A Google

Atributos De Los Enlaces

target=""

indica donde mostrar el contenido del link presionado.

target="_self": sirve para abrir el recurso en la misma pestaña y es el valor q viene por defecto.

target="_blank": sirve para abrir el recurso en una pestaña diferente

Nota: investigar este atributo y el como afecta a la etiqueta frame.

download

es un atributo booleano, si existe será un link de descarga y si no existe será un link normal. Ejemplo:

Descargar Imagen De Un Gato

NOTA: el recurso a descargar debe de existir en el "servidor".

Navegación Por Anclas

La navegacion con anclas sirve para crear un enlace que nos posicione en un punto específico de una página web.

Para utilizarlo, se necesita colocar el atributo id, en la etiqueta que contiene el punto en el q nos queremos posicionar, mientras que al enlace, en el atributo href hay q colocar el valor del atributo id precedido de una almoadilla (#). EJemplo:

Colocar id:

<h3 id="rutas-relativas">Rutas Relativas</h3>

Nombrar el href:

<a href="#rutas-relativas">Ir A Rutas Relativas</a>
