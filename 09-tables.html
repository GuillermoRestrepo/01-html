TABLAS EN HTML

Las tablas en HTML permiten organizar datos en filas y columnas. Se usan comúnmente para mostrar información estructurada, como horarios, reportes o datos tabulares.

Etiquetas

<table>:
Sirve para crear una tabla, contiene todo lo referente a esta misma.
<tr>:
(Table row), definir filas.
td:
(Table data), definir las celdas/columnas de cada fila.
<caption>:
Definir el título de una tabla
<thead>:
Definir la cabecera de una tabla "título de cada columna", utilizar <tr> para las filas, y <th> para las celda/columnas.
<tbody>:
Encapsular el contenido principal de la tabla, utilizar <tr> para las filas, y <td> para las celda/columnas.
<tfoot>:
Encapsular datos adicionales, como total de columans, notas, entre otras, utilizar <tr> para las filas, y <td> para las celda/columnas.
<colgroup>:
Permite identificar las columnas de una tabla, dentro de esta irá, tantas etiquetas <col> como el número de columnas se desea identificar.

Atributos

rowspan=""
  • Atributo valido para las etiquetas <td> y <th>.
  • Hace que una celda ocupe más de una fila
  • El valor del atributo debe de ser el número de filas a ocupar por la celda
  • Asegurarse que la tabla no se rompa, eliminando el lugar que ocupan en las siguientes filas.
colspan=""
  • Atributo valido para las etiquetas <td> y <th>.
  • Hace que una celda ocupe más de una columna
  • El valor del atributo debe de ser el número de columnas a ocupar por la celda
  • Asegurarse que la tabla no se rompa, eliminando el lugar que ocupan en las siguientes columnas.
span=""
  • Atributo valido para la etiqueta <col>.
  • Hace que una etiqueta <col> pueda abarcar más columnas
  • El valor del atributo debe de ser el número de columnas que se desea cubrir con la etiqueta <col>
  • Util para dar estilos a una o varias columans en específico.
