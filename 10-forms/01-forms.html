Formularios En HTML

Los formularios permiten que los usuarios interactuen con el sistema, dando la oportunidad de que el usuario ingrese datos que serán procesados por el servidor. Útiles en encuestas, registros, inicios de sesión, entre otros.

Estructura

La estructura basica de un formulario está compuesta por 4 etiquetas.

<form>
Es la etiqueta que encapsula el formulario.
<label>
Sirve para escribir el nombre del campo a rellenar.
<input>
Crea el campo a rellenar.
<button>
Crea el boton que permite enviar el formulario.

Ejemplo:

            
                <form action="" method="">
                    <label for="">Campo De Ejemplo</label>
                    <input type="" id="">
                    <button type="">Enviar</button>
                </from>

Atributos

action=""
Define la URL o archivo al que se enviarán los datos.
method=""
Especifica cómo se envían los datos (GET o POST).
for=""
Indica a que a que <input> pertenece el <label>.
type=""
Sirve para especificar el tipo de contenido hay q ingresar o que acción va a realizar la etiqueta <input> y <button>.

Relacionar la etiqueta <label> con la etiqueta <input>

Esto se hace para evitar errores al llenar los formularios, anunciar que se debe de colocar en cada campo, ampliar el area cliqueable, entre otras funcionalidades.

Puede hacerce mediante 2 formas, las cuales son:

  • Colocar la etiqueta <input> dentro de la etiqueta <label>, ejemplo:

                        
                        <form>
                            <label>
                                Nombre
                                <input>
                            </label>
                            <button>Enviar</button>
                        </form>

  • Haciendo que el valor del atributo for="" de la etiqueta <label> sea igual al valor del atributo id="" de la etiqueta <input>, ejemplo.

                        
                        <form>
                            <label for="name">Nombre</label>
                            <input id="name">
                            <button>Enviar</button>
                        </form>
