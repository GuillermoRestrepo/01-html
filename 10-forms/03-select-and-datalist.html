<select> & <datalist>

Sin listas desplegables que permiten al usuario elegir una o varias opciones.

<select>

Etiquetas:

  • <select>: Crea el menú desplegable.

    Atributos:

    • name="": es como la variable q contendrá el valor de la opcion seleccionado.
    • multiple: atributo booleano, permite seleccionar más de una opcion, para esto hay q presionar Ctrl + Clic en cada nueva opción.
    • size="":en el estado de no desplegado, muestra la cantidad de opciones q se le indicque

  • <option>: Crea las opciones de la lista desplegable.

    Atributos:

    • value="": es el valor q se enviará al servidor en caso de ser una opción seleccionada.
    • selected: hace q una opción esté preseleccionada, solo se puede usar una vez en un <select> que no cuenta con el atributo multiple.
    • label="": Proporciona una etiqueta alternativa al texto visible. es decir puede utilizarce como texto de la etiqueta (revisar mejor)
    • disable: deja la opcion visible pero no es seleccionable

  • <optgroup>: Se usa dentro de un <select> para agrupar visualmente varias etiquetas <option> bajo un mismo título o categoría.

    Atributos:

    • label="": establece el nombre del grupo
    • disable: desactiva todo el grupo de opciones

<datalist>

Nota: El atributolist="" de la etiqueta <input> y el atributo id="" de la etiqueta <datalist> deberán de llevar el mismo valor.
