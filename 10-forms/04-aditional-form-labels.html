ETIQUETAS ADICIONALES DE LOS FORMULARIOS

Etiquetas q sirven para complementar el uso de formularios en html.

<fieldset> & <legend>

<fieldset>: Se usa dentro de un formulario para agrupar elementos relacionados.

<legend>: Proporciona una descripción del grupo de campos que está contenido en un <fieldset>.

Ejemplo:

Datos Personales
Informacion De Contacto

<meter>

Se usa para representar un valor dentro de un rango conocido, como una nota, el nivel de batería o el uso de espacio en disco.

Atributos:

value
Valor actual.
min
Valor mínimo.
max
Valor máximo.
low
Umbral bajo. ()
high
Umbral alto. (Valor por debajo del cual se considera "bajo".)
optimum
Valor óptimo (Valor por encima del cual se considera "alto".)

Ejemplo:

50%

<progress>

Representa el progreso de una tarea, como una descarga o carga. Muestra una barra de avance que va creciendo con el tiempo.

value
Valor actual (va de 0 al máximo).
max
Valor máximo que indica el fin del proceso.

Ejemplo:

65%

<textarea>

Se usa para crear un campo de texto multilínea, donde el usuario puede escribir texto libremente, como comentarios, mensajes, descripciones, etc.

  • name: Nombre que se usará al enviar el formulario.
  • id: Identificador único (para etiquetas
  • rows: Número de filas visibles (líneas visibles a la vez).
  • cols: Número de columnas visibles (max caracteres visibles por línea).
  • placeholder: Texto de sugerencia (como en input).
  • required: Hace obligatorio completarlo.
  • maxlength: Límite máximo de caracteres.
  • readonly: Solo lectura.
  • disabled: Desactiva el campo.
  • wrap: Controla cómo se ajusta el texto al llegar al borde (soft, hard, off).
