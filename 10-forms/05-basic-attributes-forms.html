ATRIBUTOS BÁSICOS DE LOS FORMULARIOS

algunos de estos son:

placeholder: da una pista de lo qbede introducir el usuario. Ejemplo

max / min / maxleght / minleght: establecer maximo y minimo tanto para inpust numericos como textuales.

disable: desactiva un campo, lo hace inaccesible pero sigue siendo visible y no envia el dat al momento de enviar el formulario. Ejemplo:

required: hace q un campo sea estrictamente necesario de llenar, si no se llena el formulari no se envia, es como una primera capa de seguridad.

readonly: es visible, no es modificable, si envia los datos.

hidden: investigar mas q sea el atributo no el valor de type.

selected: valido en select para marcar una opción como predeterminada.
