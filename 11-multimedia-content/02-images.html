IMÁGENES EN HTML

Las imágenes en HTML se usan para:

  • Mejorar la apariencia visual de una página.
  • Ilustrar contenido.
  • Crear banners, iconos, botones, diagramas, etc.
  • Representar contenido gráfico dinámico o decorativo.

Estructura

Las imagenes en html se representan con la etiqueta <img>>

Atributos

  • src="" Dirección de la imágen (enlace local).
  • alt="" Breve descripción que sirve en caso de q la imágen no se cargue por algún error, o para mejorar la accesibilidad para personas invidentes.

Páginas Para Convertir Las imagenes En Diferentes Formatos

  • Vectorizer / Vectormagic: ambas son excelentes para convertir en svg, pero pero son de paga.
  • Online-Convert: buena para converitir (jpg, jpeg, png, gif) en (webp), también imagenes sencillas al estilo icono en (svg).

Tipos De Formatos

Formato ¿Para Que Se Usa? Características
.jpg o .jpeg Fotos, imágenes con muchos colores Comprimido con pérdida. Buen equilibrio entre calidad y tamaño.
.png Imágenes con transparencia o más calidad Sin pérdida, mayor peso. Soporta transparencia.
.gif Animaciones simples o gráficos Hasta 256 colores, puede ser animado.
.svg Gráficos vectoriales Escalable sin pérdida de calidad. Ideal para íconos, logos.
.webp Imágenes modernas optimizadas Alta compresión, buena calidad. Más liviano que PNG o JPG.
.avif Nuevo formato eficiente Mejor compresión que WebP. No soportado por todos los navegadores aún.

Notas

  • Es aconsejable q toda imágen (.jpg / .jpeg / .png / .gif), sean convertidas a formato (.webp), ya qe esto disminuye el peso de la imágen.
  • Es aconsejable que toda imagen q no tenga mucho contenido, sea convertida a formato (.svg), su fdefinición es escalable, es decir, no pierde calidad y su peso es minimo, pero solo s epuede hacer con imagenes q sean simples.
