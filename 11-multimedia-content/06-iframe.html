ETIQUETA <iframe>

La etiqueta <iframe> (inline frame) sirve para incrustar otro documento HTML dentro del actual.

Es como una ventanita que muestra otra página web dentro de tu propia página.

Sirve para:

  • Mostrar otras páginas web dentro de la tuya.
  • Incrustar videos de YouTube, mapas de Google, formularios externos, etc.
  • Insertar documentos PDF, presentaciones o contenidos de otras plataformas.
  • Mostrar contenido externo sin redirigir al usuario.

Ejemplo

Este ejemplo cargará un vide de yutube en esta página web.

Código:


            <iframe width="560" height="315"
                src="https://www.youtube.com/embed/9ThO76peOw0?si=7PwNoFwEJOVWo5AC"
                title="YouTube video player"
                frameborder="0"
                allow="accelerometer; autoplay; clipboard-write; encrypted-media;
                    gyroscope; picture-in-picture; web-share"
                referrerpolicy="strict-origin-when-cross-origin"
                allowfullscreen></iframe>

Como se muestra:

Atributos

Atributo ¿Qué Hace?
src URL del contenido que querés mostrar.
width / height Tamaño del iframe.
title Texto accesible para describir el contenido (muy importante para accesibilidad).
loading Controla la carga: lazy (carga cuando entra en pantalla) o eager.
allowfullscreen Permite que el contenido del iframe se muestre en pantalla completa.
frameborder (obsoleto) Define si se muestra un borde (0 o 1). Mejor usar CSS.
sandbox Restringe funcionalidades del contenido embebido (mejora la seguridad).
allow Define permisos como micrófono, cámara, geolocalización, etc.
referrerpolicy Controla cómo se envía el encabezado de referencia.

Nota Sobre Su Uso

En la mayoría de las ocaciones, se utilizará la etiqueta <iframe> extraida directamete de otro sitio web.
