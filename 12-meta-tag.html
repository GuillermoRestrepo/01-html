ETIQUETA <meta>

La etiqueta <meta> sirve para definir metadatos sobre página una página web.

  • Es una etiqueta vacía, es decir, no tiene contenido ni etiqueta de cierre.
  • Esta tiene que estar ubicada dentro de la etiqueta <head>
  • Los metadatos son datos sobre referentes a otros datos.

¿Para Qué Sirve La Etiqueta <meta>?

Sirve para:

  • Definir la codificación de caracteres.
  • Controlar el comportamiento de la página (ej: recarga automática, redireccionamiento).
  • Establecer descripciones para SEO.
  • Añadir palabras clave.
  • Controlar la visualización en móviles (viewport).
  • Configurar el autor, idioma, robots, redes sociales, etc.

Atributos

Atributo Para Que Sirve
name El tipo de metadato (ej: "description", "keywords")
content El valor del metadato
charset Define la codificación del texto (ej: "UTF-8")
http-equiv Comportamiento del navegador (ej: "refresh", "X-UA-Compatible")

Ejemplos Comunes Y Utiles

  1. Codificación De Caracteres (debe ir al inicio de la etiqueta <head>)

    Define la codificación de caracteres de la página, evitando errores con acentos, símbolos, etc.

    <meta charset="UTF-8">

  2. Viewport (para diseño responsivo)

    Controla cómo se muestra el sitio en dispositivos móviles y es esencial para diseño responsive.

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

  3. Autor

    Útil para atribución, aunque no visible para el usuario.

    <meta name="author" content="NombreAutor">

  4. Descripción para SEO

    Describe el contenido de la página para buscadores (SEO), este se muestra como texto en los resultados de Google.

    <meta name="description" content="Esta es una página sobre diseño web y HTML.">

  5. Palabras clave (menos usadas hoy en SEO)

    Lista de palabras clave relacionadas con la página, hoy en día tiene muy poco peso SEO, pero sigue siendo usada a veces.

    <meta name="keywords" content="HTML, CSS, programación, web.">
