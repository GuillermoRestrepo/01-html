OPEN GRAPH PROTOCOL

Open Graph Protocol es un conjunto de etiquetas <meta> que permiten a las páginas web mostrar información sobre su contenido cuando se comparte la url de estas en algún otro sitio web o red social.

Estructura

La estructura de una propiedad del protocolo open graph, se basa en una etiqueta <meta> q contiene los atributos property y content, en los cuales pueden ir los siguientes valores.

Valores del atributo (property) Valores del atrbuto (content)
og:title El título que se mostrará en la tarjeta del enlace.
og:description Descripción breve del contenido.
og:image URL absoluta de la imagen de vista previa.
og:url Enlace canónico de la página (URL final que representa el contenido).
og:type Tipo de objeto (ej: website, article, video, music.song, etc).
og:site_name Nombre del sitio web (opcional).
og:locale Idioma del contenido (ej: es_ES para español de España).

Ejemplo Código:


            <head>
                <meta property="og:title" content="Mi increíble sitio web">
                <meta property="og:description" content="Explora artículos y recursos sobre desarrollo web.">
                <meta property="og:image" content="https://miweb.com/img/preview.jpg">
                <meta property="og:url" content="https://miweb.com/">
                <meta property="og:type" content="website">
            </head>

Nota: el valor del atributo (content) cuando el valor del atributo (property) es igual a (og:image), debe de ser una ruta absoluta que un protocolo http o https.
