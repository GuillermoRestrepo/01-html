Accesibilidad

Este tema está orientado a personas con discapacidades visuales, para q estas puedan navegar de una forma sencilla en un sitio web.

Atributo tabindex=""

Este atributo nos permite navegar mediante el uso de la tecla tabulador.

codigo


            <p> tabindex="3">Párrafo 3</p>
            <p> tabindex="2">Párrafo 2</p>
            <p> tabindex="1">Párrafo 1</p>

Presiona la tecla tabulador para ver su funcionamiento:

No solo es util para la accesibilidad, sino para comodida de usuarios que desen desplazarce de forma mas rápida por las opciones del sitio web.

Atributo aria-label=""

Este atributo permite escribir una descripcion adicional q puede ser leida por medio de una extencion de screamreader (lector de pantalla).

EJEMPLO DE EXTENCION:(ChromeVox Classic Extension)

Codigo:

<a href="#" aria-label="Leer mas sobre la guerra comercial">Leer más</a>

Atributo role

Este atributo le indica al lector de pantalla qué tipo de "componente" es ese elemento: botón, encabezado, barra de navegación, alerta, diálogo, etc.

sirve para:

  • Ayuda a que los usuarios con discapacidad entiendan mejor la estructura y funcionalidad de una página web.
  • Hace accesibles elementos que no lo son de forma natural (como <div> o <span> usados con JS).
  • Refuerza el significado semántico cuando los elementos HTML no lo comunican por sí solos.

            <img src="#" rol="img" alt="imagen vacia">

Por ejemplo, aquí se le indica al navegador que este elemento es una imagen y el asistente de voz se lo comunicará al usuario.
