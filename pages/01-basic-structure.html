ESTRUCTURA BÁSICA DE UN DOCUMENTO HTML

  • <!DOCTYPE html>
  • <html lang="es">
  • <head>
  • <meta charset="UTF-8">
  • <meta name="viewport">
  • <title>
  • <body>

La estructura básica de un documento HTML es la siguiente:

    <!DOCTYPE html>
    <html lang="es">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>

    </body>
    </html>

1. <!DOCTYPE html>

  • No es una etiqueta HTML, sino una declaración especial.
  • No tiene cierre.
  • Debe colocarse en la primera línea del documento.
  • Informa al navegador sobre el tipo de documento que está ejecutando.
☝️ Inicio

2. <html lang="es">

  • Es la etiqueta raíz de todo documento HTML.
  • Encierra todo el contenido del documento, excepto la declaración <!DOCTYPE html>.
  • Indica el inicio y el fin del documento HTML con <html> y </html>.
  • El atributo lang especifica el idioma principal del contenido.
  • El atributo lang ayuda a los navegadores, buscadores y lectores de pantalla a interpretar correctamente el idioma del texto.
☝️ Inicio

3. <head>

  • Es una etiqueta contenedora que agrupa metadatos y configuraciones del documento HTML.
  • Se coloca dentro de <html> y antes de <body>.
  • No contiene contenido visible directamente en la página web (excepto el título en la pestaña del navegador).
  • Su función principal es proporcionar información sobre el documento, no mostrar contenido.
  • Contiene etiquetas esenciales para el correcto funcionamiento y presentación del sitio.
  • Ayuda al navegador, a los motores de búsqueda y a las redes sociales a interpretar correctamente el documento.
☝️ Inicio

4. <meta charset="UTF-8">

  • Pertenece al grupo de etiquetas <meta>.
  • Se coloca dentro de la etiqueta <head>.
  • No tiene etiqueta de cierre.
  • Su función principal es definir la codificación de caracteres del documento HTML.
  • El atributo charset indica el conjunto de caracteres que se usará para interpretar el texto.
  • UTF-8 (Unicode Transformation Format – 8 bits) es la codificación estándar universal recomendada por el W3C.
  • Permite representar todos los caracteres y símbolos de prácticamente todos los idiomas del mundo (acentos, eñes, letras especiales, emojis, etc.).
☝️ Inicio

5. <meta name="viewport">

  • Pertenece al grupo de etiquetas <meta>.
  • Se coloca dentro de la etiqueta <head>.
  • No tiene etiqueta de cierre.
  • Permite que las páginas sean responsivas (adaptables) y se visualicen correctamente en distintos tamaños de pantalla.
  • El atributo name="viewport" indica que la etiqueta configura la ventana gráfica (viewport) del navegador.
  • El viewport es el área visible de una página web en la pantalla del dispositivo.
  • El valor width=device-width indica que el ancho del contenido debe ajustarse al ancho del dispositivo.
  • El valor initial-scale=1.0 define que la escala inicial de la página será del 100 %, es decir, sin hacer zoom al cargar.
☝️ Inicio

6. <title>

  • Es una etiqueta obligatoria en todos los documentos HTML.
  • Define el título del documento, que aparece en la pestaña del navegador, en los marcadores (favoritos) y en los resultados de búsqueda.
  • Se coloca dentro de la etiqueta <head>.
  • Solo puede haber una por documento HTML.
  • No admite etiquetas HTML dentro de ella, solo texto plano.
  • El texto contenido entre las etiquetas <title> y </title> representa el nombre oficial del documento.
☝️ Inicio

7. <body>

  • Representa el cuerpo del documento HTML.
  • Es una etiqueta contenedora que agrupa todo el contenido visible de la página web.
  • Se coloca dentro de la etiqueta <html> y después de la sección <head>.
  • Solo puede haber una por documento.
  • Puede contener todo tipo de elementos estructurales, semánticos y multimedia.
