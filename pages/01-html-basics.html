Nombrar Archivos (.html)

Caracteres Permitidos

Los nombres para archivos HTML pueden contener:

  • Letras minúsculas.
  • Números.
  • Guiones medios para reemplazar espacios (kebab-case).

Caracteres A Evitar

Es importante evitar el uso de:

  • Mayúsculas.
  • Caracteres especiales y símbolos. (ñ, /, *, @, ?, !, etc.).
  • Signos de puntuación (,, ., :, ;, etc.).
  • Acentos (á, ê, ü, etc.).

Ejemplos

Forma correcta:

  • index.html
  • mi-pagina-2.html
  • 01-basicos.html

Forma incorrecta:

  • índice.html
  • mí_página(2).html
  • 01 basicos.html

Comentarios

Para representar comentarios en un documento HTML, hay que encerrar el comentario entre los siguientes símbolos:

  • Apertura: <!--
  • Cierre: -->

Ejemplos:

Comentario de línea:

    <!-- Esto es un comentario de línea. -->

Comentario de bloque:

    <!--
        Esto es
        un comentario
        de bloque.
    -->

Sintaxis

Existen dos tipos de etiquetas en HTML, las cuales son:

Etiqueta De Apertura

Se representa escribiendo el nombre de la etiqueta entre signos de mayor que (<) y menor que (>):

    <nombre>

Etiqueta De Cierre

Se representa de igual forma que una etiqueta de apertura, pero con la diferencia de que una etiqueta de cierre lleva un carácter de barra diagonal (/) antes del nombre:

    </nombre>

Tipos De Etiquetas

Según Su Estructura

Etiquetas de apertura y cierre

Son etiquetas que, para representarlas, es necesario escribir tanto su etiqueta de apertura como su etiqueta de cierre. Ejemplo:

    <p>Contenido de la etiqueta.</p>

Etiquetas autocerradas

Son etiquetas que, para representarlas, únicamente es necesario escribir la etiqueta de apertura. Ejemplo:

    <img>

Según su comportamiento

Etiquetas de bloque

Se caracterizan por ocupar todo el ancho disponible del elemento que las contiene y por generar un salto de línea. Ejemplo:

    <div>Elemento de bloque</div>

Etiquetas de línea

Se caracterizan por ocupar únicamente el ancho que necesita su contenido. Ejemplo:

    <span>Elemento de línea</span>
