Estructura básica de un documento HTML

La estructura básica de un documento HTML es:

    <!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Document</title>
    </head>
    <body>
        
    </body>
    </html>

<!DOCTYPE>

  • Es una declaración, no una etiqueta.
  • Se coloca al principio del documento HTML.
  • Indica al navegador la versión de HTML que utiliza el documento.

<html lang="en">

  • Es la etiqueta raíz de un documento HTML.
  • Contiene todos los demás elementos.
  • lang="" es un atributo que sirve para indicar el idioma del contenido de una etiqueta.

<head>

Sirve para contener información referente al documento HTML, también conocidos como metadatos.

<meta charset="UTF-8">

Define la codificación de caracteres del documento HTML y permite mostrar correctamente los caracteres especiales, acentos y símbolos.

<meta name="viewport">

Ayuda a que la página web se adapte y se visualice correctamente en dispositivos móviles.

<title>

Define el título del documento HTML. Se muestra en:

  • La pestaña del navegador.
  • Los resultados de búsqueda.
  • Los marcadores.
  • El historial de navegación.

<body>

Contiene el contenido visible de la página web, como texto, imágenes, enlaces, tablas y otros elementos.
