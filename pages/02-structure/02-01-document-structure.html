Estructura básica de un documento HTML

La estructura básica de un documento HTML contiene los siguientes elementos:

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título de la página</title>
</head>
<body>

</body>
</html>

1. <!DOCTYPE html>

¿Qué es?

No es una etiqueta HTML.

Es una declaración que le dice al navegador:

«Este documento utiliza HTML5.»

Debe ir siempre en la primera línea del documento.

¿Por qué es importante?

Sin esta declaración, el navegador puede entrar en modo quirks (modo de compatibilidad heredada) y el diseño puede comportarse de manera inesperada.

Siempre debe escribirse exactamente así:

<!DOCTYPE html>

No lleva etiqueta de cierre.

2. <html>

Es el elemento raíz del documento.

Todo el contenido debe estar dentro de esta etiqueta.

<html lang="es">

El atributo lang

Indica el idioma principal del contenido.

Ejemplos:

  • <html lang="es">
  • <html lang="en">
  • <html lang="fr">

Esto mejora:

  • La accesibilidad
  • El SEO
  • El funcionamiento de los lectores de pantalla

3. <head>

Contiene información sobre el documento.

Nada dentro del <head> se muestra como contenido visual de la página.

Elementos importantes dentro del <head>

<meta charset="UTF-8">

Define la codificación de caracteres.

<meta charset="UTF-8">

Permite usar:

  • Tildes
  • La letra ñ
  • Símbolos especiales

Siempre debe colocarse al inicio del <head>.

<meta name="viewport">

Es esencial para el diseño responsive.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Permite que la página se adapte correctamente en dispositivos móviles.

Actualmente es un estándar en proyectos reales.

<title>

Define el título que aparece en:

  • La pestaña del navegador
  • Los resultados de búsqueda
<title>Mi página web</title>

Debe existir siempre en todo documento HTML.

<link> (para CSS)

Permite conectar hojas de estilo externas.

<link rel="stylesheet" href="estilos.css">

<script> (para JavaScript)

Se utiliza para incluir código JavaScript. Puede colocarse dentro del <head> o antes de cerrar la etiqueta <body>.

4. <body>

Aquí va todo el contenido visible de la página.

Ejemplo:

<body>
    <h1>Hola mundo</h1>
    <p>Bienvenido a mi sitio web.</p>
</body>
