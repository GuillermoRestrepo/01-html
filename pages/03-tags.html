ETIQUETAS

1. Tipos De Etiquetas

Existen dos tipos de etiquetas en HTML:

1.1. Etiquetas De Apertura y Cierre

  • Son el tipo de etiqueta más común en HTML.
  • Tienen dos partes:
    1. Etiqueta de apertura (<nombre>).
    2. Etiqueta de cierre (</nombre>).
  • Pueden contener texto, otras etiquetas o ambas.
  • La etiqueta de cierre siempre lleva /.

Ejemplo

Código Resultado 
<p>
    Esta es una etiqueta
    de apertura y cierre.
</p>

Esta es una etiqueta de apertura y cierre.

1.2. Etiquetas Autocerradas

  • Son etiquetas que no tienen contenido.
  • No tienen etiqueta de cierre.
  • Su función es insertar o declarar algo.

Ejemplo

Código Resultado 
<p>
    <code>&lt;br&gt;</code>
    es una etiqueta<br>
    autocerrada.
</p>

<br> es una etiqueta
autocerrada.

2. Etiquetas De Bloque Y Etiquetas De Línea

Esta clasificación se basa en cómo se muestran visualmente las etiquetas y cómo ocupan el espacio en la página.

2.1. Etiquetas de bloque

  • Comienzan en una nueva línea.
  • Por defecto, generan un salto de línea.
  • Ocupan todo el ancho disponible.
  • Pueden contener otras etiquetas de bloque y de línea.

Ejemplo

Código Resultado 
<p>Esto es un párrafo.</p>
<p>Esto es otro párrafo.</p>

Esto es un párrafo.

Esto es otro párrafo.

2.2. Etiquetas de línea

  • No rompen la línea.
  • No generan salto de línea.
  • Ocupan solo el espacio de su contenido.
  • Se colocan una junto a otra.

Ejemplo

Código Resultado 
<p>
    Esto es <em>importante</em>,
    pero esto otro es más <strong>importante</strong>.
</p>

Esto es importante, pero esto otro es más importante.
