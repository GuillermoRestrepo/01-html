Los atributos son propiedades o características adicionales que se añaden a las etiquetas HTML para modificar su comportamiento, cambiar su aspecto o aportar información adicional.

Sintaxis básica

La estructura de un atributo es la siguiente:

<etiqueta atributo="valor">Contenido</etiqueta>

Ejemplo:

<a href="https://www.google.com">Ir a Google</a>
<a> Etiqueta.
href Atributo.
"https://www.google.com" Valor del atributo.
Ir a Google Contenido de la etiqueta.

Tipos de atributos

Atributos globales

Pueden usarse en cualquier etiqueta HTML. Algunos de ellos son:

Atributo Descripción Ejemplo
id Identifica de forma única un elemento. <div id="principal">
class Agrupa elementos para CSS o JavaScript. <p class="texto">
style Aplica estilos en línea. <h1 style="color:red;">Hola</h1>
title Muestra un texto emergente al pasar el mouse. <abbr title="Organización de las Naciones Unidas">ONU</abbr>
lang Define el idioma del contenido. <p lang="es">Hola</p>
dir Dirección del texto (ltr o rtl). <p dir="rtl">Texto árabe</p>
hidden Oculta el elemento (booleano). <p hidden>Texto oculto</p>
tabindex Orden de tabulación. <input tabindex="1">
contenteditable Permite editar el texto del elemento. <div contenteditable="true">Editar</div>
data-* Crea atributos personalizados. <div data-user="123">

Atributos específicos (por etiqueta)

Estos solo funcionan con determinadas etiquetas. Algunos de ellos son:

Etiqueta Atributo Descripción Ejemplo
<a> target Define dónde abrir el enlace (_blank, _self, etc.). <a target="_blank">
<img> src Ruta o fuente de la imagen. <img src="foto.png">
<input> placeholder Texto de ayuda. <input placeholder="Tu nombre">
<form> method Método de envío (GET o POST). <form method="POST">

Atributos booleanos

No necesitan valor; su sola presencia indica “verdadero”. Algunos de ellos son:

  • checked (en <input>)
  • disabled (en <button>, <input>)
  • readonly
  • required
  • autofocus
  • multiple
  • hidden
  • selected

Atributos personalizados (data-*)

  • Permiten almacenar información adicional en los elementos sin afectar su funcionalidad.
  • Son útiles para JavaScript y el manejo dinámico de datos.
  • Se escriben con el prefijo data- seguido de un nombre definido por el desarrollador.

Ejemplo:

<div data-usuario="42" data-rol="admin">Guillermo</div>

Acceso mediante JavaScript:

const div = document.querySelector('div');
console.log(div.dataset.usuario); // 42
console.log(div.dataset.rol); // admin
