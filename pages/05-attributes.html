ATRIBUTOS EN HTML

Los atributos son palabras o valores adicionales que se añaden a las etiquetas para configurar el comportamiento o la funcionalidad de un elemento.

Sintaxis de los atributos

Existen dos tipos de atributos principales:

Atributos con nombre y valor

Estos atributos se representan mediante el nombre del atributo y el valor que este puede tener, escrito entre comillas. Ejemplo:

<p class="parrafo">

Atributos booleanos

Estos atributos se representan escribiendo únicamente el nombre del atributo. Si están presentes, se aplican; si no existen, no tienen efecto. Ejemplo:

<video autoplay>

Atributos globales

Son atributos que pueden ser utilizados en todas o casi todas las etiquetas. Algunos ejemplos son:

  • id: Identificador único.
  • class: Clase (para CSS o JS).
  • style: Estilos CSS en línea.
  • title: Texto que aparece como tooltip al pasar el cursor.
  • lang: Idioma del contenido.
  • dir: Dirección del texto (ltr, rtl).
  • hidden: Oculta un elemento, aunque sigue existiendo en el DOM.
  • tabindex: Define el orden de tabulación y si el elemento puede recibir foco.
  • contenteditable: Permite editar el contenido en el navegador.
  • draggable: Hace que un elemento sea arrastrable.
  • data-*: Atributos personalizados para almacenar datos.

Atributos específicos

Son atributos que solo pueden utilizarse en determinadas etiquetas. Ejemplo:

<form method="post">
