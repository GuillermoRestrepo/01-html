LISTAS EN HTML

  • Listas ordenadas

  • Listas de definición
  • Listas anidadas

Las listas sirven para organizar contenido en forma de elementos agrupados.

1. Listas ordenadas

  • Sirven para representar listas cuyos elementos sí tienen un orden específico.
  • Se definen con la etiqueta <ol> (Ordered List).
  • Los elementos se definen con la etiqueta <li>.

Atributos de las listas ordenadas

Atributo Valores
type 1 (predeterminado), i, I, a, A
start Número desde el cual inicia la lista.
reversed Indica si la lista se muestra en orden inverso.

Ejemplo

Código Resultado 
<ol start="4" type="I" reversed>
    <li>Elemento 1</li>
    <li>Elemento 2</li>
    <li>Elemento 3</li>
</ol>
  1. Elemento 1
  2. Elemento 2
  3. Elemento 3

2. Listas desordenadas

  • Sirven para representar listas cuyos elementos no tienen un orden específico.
  • Se definen con la etiqueta <ul> (Unordered List).
  • Los elementos se definen con la etiqueta <li>.

Atributos de las listas desordenadas

Atributo Valores
type disc (predeterminado), circle, square

Ejemplo

Código Resultado 
<ul type="circle">
    <li>Elemento 1</li>
    <li>Elemento 2</li>
    <li>Elemento 3</li>
</ul>
  • Elemento 1
  • Elemento 2
  • Elemento 3

3. Listas de definición

  • Sirven para representar listas cuyos elementos tienen conceptos y descripciones asociadas.
  • Se definen con la etiqueta <dl> (Definition List).
  • Los términos se definen con <dt> (Definition Term).
  • Las descripciones se definen con <dd> (Definition Description).

Ejemplo

Código Resultado 
<dl>
    <dt>API</dt>
    <dd>Interfaz para la comunicación entre sistemas.</dd>

    <dt>DOM</dt>
    <dd>Estructura jerárquica del documento HTML.</dd>
</dl>
API
Interfaz para la comunicación entre sistemas.
DOM
Estructura jerárquica del documento HTML.

4. Listas anidadas

Son listas que contienen otras listas dentro de sus elementos.

Ejemplo

Código Resultado 
<ul>
    <li>Frontend
        <ol>
            <li>HTML</li>
            <li>CSS</li>
            <li>JavaScript</li>
        </ol>
    </li>
    <li>Backend</li>
    <li>Base de datos</li>
</ul>
  • Frontend
    1. HTML
    2. CSS
    3. JavaScript
  • Backend
  • Base de datos
