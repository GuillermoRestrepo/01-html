ETIQUETAS DE TEXTO I

1. <h1> … <h6>

  • Representan títulos y subtítulos del contenido.
  • Definen la jerarquía y estructura del documento.
  • Van desde mayor (<h1>) a menor (<h6>) importancia.
  • Ayudan a la organización del contenido.
  • Solo puede haber un <h1> por documento.

2. <p>

  • Representa un párrafo de texto.
  • No debe contener otras etiquetas de bloque.

3. <small>

  • Representa texto de menor importancia.
  • Se usa para notas, aclaraciones o advertencias.
  • Reduce visualmente el tamaño del texto.
  • Tiene significado semántico.

4. <em>

  • Indica énfasis en el contenido.
  • Cambia el significado del texto al resaltarlo.
  • Es una etiqueta semántica.
  • Normalmente se muestra en cursiva.

5. <strong>

  • Indica fuerte importancia.
  • Resalta contenido relevante o crítico.
  • Es una etiqueta semántica.
  • Normalmente se muestra en negrita.

6. <i>

  • Representa texto en una voz alternativa.
  • Se usa para términos técnicos, nombres o pensamientos.
  • No aporta una importancia semántica fuerte.
  • Normalmente se muestra en cursiva.

7. <b>

  • Aplica resaltado visual sin semántica.
  • No indica importancia ni énfasis.
  • Se usa solo con fines de presentación.
  • Normalmente se muestra en negrita.

8. <u>

  • Representa anotaciones o marcas no textuales.
  • No se usa únicamente para subrayar por estilo.
  • Puede causar confusión con enlaces.
  • Su uso es limitado y específico.

9. <sub>

  • Representa un subíndice.
  • Se usa en fórmulas químicas o matemáticas.
  • Reduce el tamaño y baja el texto.
  • Mantiene significado semántico.

10. <sup>

  • Representa un superíndice.
  • Se usa para exponentes o referencias.
  • Reduce el tamaño y eleva el texto.
  • Mantiene significado semántico.

11. <del>

  • Representa contenido eliminado.
  • Indica cambios o correcciones.
  • Se muestra como texto tachado.
  • Tiene significado semántico.

12. <ins>

  • Representa contenido insertado.
  • Se usa junto con <del>.
  • Indica modificaciones recientes.
  • Tiene significado semántico.
