ENLACES (FUNDAMENTOS)

Los enlaces en HTML permiten conectar un documento con otro recurso, ya sea otra página web, un archivo dentro del mismo proyecto o incluso un lugar específico dentro de la misma página. Su propósito principal es la navegación entre documentos.

La estructura básica de un enlace utiliza la etiqueta <a>, que envuelve el texto o elemento en el cual el usuario puede hacer clic. 

<a href="ruta/del/recurso">Texto del enlace</a>

Tipos de rutas

Rutas relativas

Son aquellas que indican la ubicación de un recurso en relación con la posición del archivo HTML actual. Para navegar entre carpetas se utilizan:

  • / para entrar en una carpeta.
  • ../ para retroceder a la carpeta anterior.

Ejemplos:

<a href="paginas/contacto.html">Ir a contacto</a>
<a href="../index.html">Regresar al inicio</a>

Rutas absolutas

Son aquellas que indican la dirección completa de un recurso, sin depender de la ubicación del archivo HTML actual. Se utilizan generalmente para acceder a recursos en internet.

Ejemplo:

<a href="https://www.ejemplo.com">Visitar Ejemplo</a>
