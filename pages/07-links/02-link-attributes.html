ATRIBUTOS DE LOS ENLACES

href

El atributo href define la dirección del recurso al cual apunta el enlace.

Ejemplos:

<a href="pagina.html">Enlace a una página local</a>
<a href="https://www.ejemplo.com">Enlace externo</a>

target

El atributo target indica dónde se abrirá el recurso vinculado.

Valores posibles:

  • _self: Abre en la misma pestaña (valor por defecto).
  • _blank: Abre en una pestaña o ventana nueva.
  • _parent: Abre en el marco padre.
  • _top: Abre en toda la ventana, ignorando marcos.

Ejemplo:

<a href="pagina.html" target="_blank">Abrir en nueva pestaña</a>

download

El atributo download indica que, en lugar de abrir el recurso, este debe descargarse. Opcionalmente, se puede especificar el nombre del archivo.

Ejemplos:

<a href="documento.pdf" download>Descargar documento</a>
<a href="imagen.png" download="foto-usuario.png">Descargar imagen</a>

hreflang

El atributo hreflang especifica el idioma del recurso enlazado, lo cual es útil para navegadores y motores de búsqueda.

Ejemplo:

<a href="pagina-en.html" hreflang="en">Versión en inglés</a>

rel

El atributo rel define la relación entre el documento actual y el recurso enlazado.

Algunos valores comunes son:

  • noopener: Evita que la nueva página acceda a window.opener.
  • noreferrer: Evita enviar información de referencia.
  • nofollow: Indica a los motores de búsqueda que no sigan el enlace.
  • alternate: Indica que el recurso es una versión alternativa.
  • stylesheet: Enlaces a hojas de estilo.

Ejemplo:

<a href="https://www.ejemplo.com" rel="nofollow">Enlace con nofollow</a>

type

El atributo type indica el tipo de medio (MIME type) del recurso al que se enlaza. Es más usado en enlaces a recursos distintos a páginas HTML.

Ejemplos:

<a href="documento.pdf" type="application/pdf">Abrir PDF</a>
<a href="estilos.css" rel="stylesheet" type="text/css">Enlazar CSS</a>
