USOS ESPECIALES DE LOS ENLACES

Además de enlazar a páginas web o documentos, los enlaces en HTML pueden usarse para ejecutar acciones especiales como enviar correos, realizar llamadas telefónicas o abrir aplicaciones externas. Esto se logra utilizando esquemas específicos en el atributo href.

Correo electrónico: mailto:

Permite abrir el cliente de correo del usuario para redactar un nuevo mensaje a una dirección determinada. 

<a href="mailto:ejemplo@correo.com">Enviar correo</a>

Incluso se pueden agregar parámetros como asunto o cuerpo del mensaje: 

<a href="mailto:ejemplo@correo.com?subject=Consulta&body=Hola">Correo con asunto</a>

Llamadas telefónicas: tel:

Abre la aplicación de teléfono en dispositivos compatibles para realizar una llamada al número indicado. 

<a href="tel:+593987654321">Llamar al número</a>

WhatsApp

Se puede abrir un chat de WhatsApp utilizando un enlace especial con el número de teléfono y, opcionalmente, un mensaje predefinido. 

<a href="https://wa.me/593987654321">Abrir chat en WhatsApp</a>
<a href="https://wa.me/593987654321?text=Hola%20quiero%20información">Enviar mensaje por WhatsApp</a>

Otros esquemas comunes

Enlace a ubicación (Google Maps)

<a href="https://www.google.com/maps?q=-2.170998, -79.922359">Ver ubicación</a>

Enlace para descargar un archivo directo

<a href="https://ejemplo.com/archivo.pdf">Descargar PDF</a>
