ETIQUETAS DE TEXTO II

1. <hr>

  • Representa una separación temática.
  • Es una etiqueta autocerrada.
  • Se muestra como una línea horizontal.
  • Tiene significado semántico.

2. <br>

  • Inserta un salto de línea.
  • Es una etiqueta autocerrada.
  • No debe usarse para separar secciones.
  • Su uso es puntual dentro de texto.

3. <wbr>

  • Indica un punto de quiebre de línea opcional.
  • El navegador decide si rompe la línea.
  • Es una etiqueta autocerrada.
  • Es útil para palabras largas.

4. <time>

  • Representa fechas u horas.
  • Mejora la semántica del contenido.
  • Puede incluir el atributo datetime.
  • Es útil para eventos y publicaciones.

5. <address>

  • Representa información de contacto.
  • Se asocia al autor o propietario del contenido.
  • No se usa para direcciones genéricas.
  • Tiene significado semántico.

6. <span>

  • Es un contenedor en línea genérico.
  • No aporta semántica por sí mismo.
  • Se usa para CSS y JavaScript.
  • No genera salto de línea.

7. <div>

  • Es un contenedor de bloque genérico.
  • No aporta semántica por sí mismo.
  • Se usa para estructurar y agrupar contenido.
  • Genera salto de línea.

8. <pre>

  • Representa texto preformateado.
  • Respeta espacios y saltos de línea.
  • Usa tipografía monoespaciada.
  • Es una etiqueta de bloque.

9. <code>

  • Representa código fuente.
  • Usa tipografía monoespaciada.
  • Es una etiqueta en línea.
  • Puede combinarse con <pre>.

10. <q>

  • Representa una cita corta.
  • Se muestra entre comillas automáticamente.
  • Es una etiqueta en línea.
  • Tiene significado semántico.

11. <blockquote>

  • Representa una cita extensa.
  • Es una etiqueta de bloque.
  • Puede incluir el atributo cite.
  • Tiene significado semántico.

12. <kbd>

  • Representa la entrada del usuario.
  • Se usa para teclas o comandos.
  • Usa tipografía monoespaciada.
  • Tiene significado semántico.
