LISTAS

Las listas en HTML permiten organizar y presentar información de manera estructurada. Existen diferentes tipos de listas según el orden y la relación entre los elementos.

Listas ordenadas (<ol>)

Se utilizan cuando los elementos deben seguir un orden específico. Cada ítem se marca con un número o un estilo de enumeración. Sus atributos principales son:

  • type: Define el tipo de marcador (1, A, a, I, i).
  • start: Define el número o valor inicial de la lista.
  • reversed: Invierte el orden de la numeración.

Ejemplo

<ol type="A" start="3" reversed>
    <li>Primer elemento</li>
    <li>Segundo elemento</li>
    <li>Tercer elemento</li>
</ol>
  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento

Listas desordenadas (<ul>)

Se utilizan cuando los elementos no necesitan seguir un orden específico. Los ítems aparecen con viñetas como marcadores. Sus atributos principales son:

  • type: Permite definir el estilo de viñeta (disc, circle, square). Aunque este atributo está en desuso, aún funciona en algunos navegadores.

Ejemplo

<ul type="circle">
    <li>Elemento uno</li>
    <li>Elemento dos</li>
    <li>Elemento tres</li>
</ul>
  • Elemento uno
  • Elemento dos
  • Elemento tres

Listas de definición (<dl>)

Se utilizan para representar pares de términos y sus descripciones. No llevan atributos propios, pero se componen de:

  • <dt>: Define el término.
  • <dd>: Define la descripción o definición del término.

Ejemplo

<dl>
    <dt>HTML</dt>
    <dd>Lenguaje de marcado para estructurar documentos web.</dd>
        
    <dt>CSS</dt>
    <dd>Lenguaje para dar estilo y diseño a documentos HTML.</dd>
        
    <dt>JavaScript</dt>
    <dd>Lenguaje de programación para añadir interactividad.</dd>
</dl>
HTML
Lenguaje de marcado para estructurar documentos web.
CSS
Lenguaje para dar estilo y diseño a documentos HTML.
JavaScript
Lenguaje de programación para añadir interactividad.
