LISTAS EN HTML

Se utilizan para agrupar elementos relacionados y mostrarlos de forma ordenada, desordenada o descriptiva.

Existen tres tipos de listas:

1. Listas Desordenadas (<ul>)

  • Representan una lista sin un orden específico.
  • Los elementos aparecen con viñetas.
  • El orden de los ítems no altera el significado.

Ejemplo

<ul>
    <li>Elemento</li>
    <li>Elemento</li>
</ul>

2. Listas Ordenadas (<ol>)

  • Representan una lista con un orden específico.
  • Los elementos se numeran automáticamente.
  • El orden de los ítems sí importa.

Ejemplo

<ol>
    <li>Paso uno</li>
    <li>Paso dos</li>
</ol>

3. Listas De Definición (<dl>)

  • Representan pares de término y definición.
  • No utilizan viñetas ni numeración.

Ejemplo

<dl>
    <dt>Término</dt>
    <dd>Definición</dd>
</dl>
