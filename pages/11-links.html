ENLACES

1. La Etiqueta <a>

La etiqueta <a> se utiliza para crear enlaces en HTML y permite conectar documentos, recursos o secciones.

<a href="destino.html">Texto del enlace</a>
  • <a> es la etiqueta de anclaje.
  • href define el destino del enlace.
  • El contenido del enlace es el texto clickeable.

2. Tipos De Enlaces

2.1. Enlaces Externos

Apuntan a sitios web externos.

<a href="https://www.ejemplo.com">Sitio externo</a>

2.2. Enlaces Internos

Conectan páginas dentro del mismo sitio web.

<a href="contacto.html">Contacto</a>

2.3. Enlaces A Secciones

Permiten navegar a una sección específica de la misma página.

<a href="#seccion">Ir a sección</a>

3. Atributos Importantes Del Enlace

3.1. href

Define el destino del enlace y es obligatorio.

3.2. target

Indica dónde se abrirá el enlace.

<a href="https://www.ejemplo.com" target="_blank">Abrir en nueva pestaña</a>

3.3. rel

Describe la relación entre la página actual y el destino del enlace.

<a href="https://www.ejemplo.com" target="_blank" rel="noopener noreferrer">Enlace seguro</a>
