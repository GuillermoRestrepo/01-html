FORMULARIOS EN HTML

Un formulario HTML tiene una estructura básica que permite capturar datos del usuario y enviarlos correctamente.

1. Etiqueta Principal: <form>

  • Es el contenedor principal del formulario.
  • Todo campo de entrada debe estar dentro de esta etiqueta.
<form>
</form>

Funciones Principales

  • Agrupa los campos del formulario.
  • Define cómo y a dónde se envían los datos.

2. Atributos Esenciales De <form>

Aunque no son obligatorios, estos atributos son fundamentales para el funcionamiento del formulario.

action

  • Indica a dónde se envían los datos.
  • Puede ser un archivo, una URL o una ruta.
<form action="procesar.php">
</form>

method

  • Define cómo se envían los datos.
  • Valores principales: get y post.
<form method="post">
</form>

3. Campos De Entrada: <input>

Es la etiqueta más utilizada para capturar datos del usuario. 

<input type="text">

Características

  • Es una etiqueta autocerrada.
  • Su comportamiento depende del atributo type.

4. Etiqueta <label>

Describe el campo de entrada.

<label for="nombre">Nombre</label>

Funciones

  • Asocia texto descriptivo al campo.
  • Mejora la accesibilidad.
  • Permite enfocar el campo al hacer clic en el texto.

5. Asociación label + input

La asociación se realiza utilizando los atributos for e id. 

<label for="nombre">Nombre</label>
<input type="text" id="nombre" name="nombre">

6. Etiqueta <button>

Permite enviar el formulario o ejecutar acciones.

<button type="submit">Enviar</button>

Tipos Comunes

  • submit
  • reset
  • button
