INPUT DE FECHA Y HORA EN HTML

El atributo type del elemento <input> define el tipo de dato que el usuario puede ingresar. HTML incluye varios tipos específicos para trabajar con fechas y horas.

1. type="date"

  • Permite seleccionar una fecha.
  • Incluye día, mes y año.
  • Muestra un selector de calendario.
<label for="fecha">Fecha:</label>
<input type="date" id="fecha" name="fecha">

2. type="time"

  • Permite seleccionar una hora.
  • Incluye horas y minutos.
  • Puede incluir segundos según el navegador.
<label for="hora">Hora:</label>
<input type="time" id="hora" name="hora">

3. type="datetime-local"

  • Permite seleccionar fecha y hora.
  • No incluye zona horaria.
  • Es el reemplazo moderno de datetime.
<label for="fecha-hora">Fecha y hora:</label>
<input type="datetime-local" id="fecha-hora" name="fecha_hora">

4. type="month"

  • Permite seleccionar un mes y un año.
  • No incluye el día.
  • Útil para periodos mensuales.
<label for="mes">Mes:</label>
<input type="month" id="mes" name="mes">

5. type="week"

  • Permite seleccionar una semana del año.
  • Incluye el número de semana y el año.
  • Formato basado en el estándar ISO.
<label for="semana">Semana:</label>
<input type="week" id="semana" name="semana">

Nota Sobre type="datetime"

El valor datetime está obsoleto y no debe usarse. En su lugar, se recomienda datetime-local.
