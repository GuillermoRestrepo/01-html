INPUTS OPTIMIZADOS PARA DISPOSITIVOS MÓVILES

HTML ofrece ciertos valores del atributo type en la etiqueta <input> que están especialmente pensados para dispositivos móviles. Estos tipos hacen que el navegador muestre un teclado adecuado (numérico, correo, teléfono, etc.), mejorando la experiencia del usuario.

1. type="search"

  • Se utiliza para campos de búsqueda.
  • Muestra un teclado optimizado para búsquedas.
  • Algunos navegadores añaden un botón para borrar el texto.
<input type="search" name="busqueda">

2. type="email"

  • Diseñado para direcciones de correo electrónico.
  • Muestra un teclado con el símbolo @ y ..
  • Incluye validación automática del formato.
<input type="email" name="correo">

3. type="number"

  • Permite ingresar solo valores numéricos.
  • Muestra un teclado numérico en móviles.
  • Puede usar atributos como min, max y step.
<input type="number" name="edad" min="0" max="100">

4. type="tel"

  • Usado para números telefónicos.
  • Muestra un teclado numérico.
  • No valida el formato automáticamente.
<input type="tel" name="telefono">

5. type="password"

  • Oculta el texto ingresado.
  • Se usa para contraseñas.
  • En móviles permite gestores de contraseñas.
<input type="password" name="clave">

6. type="url"

  • Diseñado para direcciones web.
  • Muestra un teclado con caracteres como / y ..
  • Incluye validación básica de URLs.
<input type="url" name="sitio_web">

7. ¿Por Qué Usar Estos Tipos?

  • Mejoran la experiencia del usuario en móviles.
  • Reducen errores de entrada.
  • Activan validaciones automáticas.
  • Facilitan el llenado de formularios.
