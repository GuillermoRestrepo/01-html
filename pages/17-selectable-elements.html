ELEMENTOS SELECCIONABLES EN HTML

Los elementos seleccionables permiten al usuario elegir una o varias opciones dentro de un formulario. HTML ofrece distintos controles según el tipo de selección que se necesite.

1. input type="radio"

Se utiliza cuando el usuario debe elegir una sola opción entre varias.

  • Las opciones se agrupan usando el atributo name.
  • Solo un radio puede estar seleccionado a la vez.
  • Se suele usar junto con <label>.
<label>
    <input type="radio" name="genero" value="m"> Masculino
</label>

<label>
    <input type="radio" name="genero" value="f"> Femenino
</label>

2. input type="checkbox"

Se utiliza cuando el usuario puede seleccionar una o varias opciones.

  • Cada checkbox funciona de forma independiente.
  • Se pueden marcar múltiples opciones.
  • Ideal para preferencias o configuraciones.
<label>
    <input type="checkbox" name="lenguajes[]" value="html"> HTML
</label>

<label>
    <input type="checkbox" name="lenguajes[]" value="css"> CSS
</label>

3. Lista desplegable: <select>

Permite seleccionar una opción desde una lista desplegable.

  • Se compone de las etiquetas <select> y <option>.
  • Por defecto solo permite una selección.
  • Puede permitir múltiples selecciones con el atributo multiple.
<select name="pais">
    <option value="">Seleccione un país</option>
    <option value="ec">Ecuador</option>
    <option value="co">Colombia</option>
</select>

4. Lista de sugerencias: <datalist>

Proporciona una lista de opciones sugeridas mientras el usuario escribe.

  • Se asocia a un <input> mediante el atributo list.
  • No obliga al usuario a elegir una opción existente.
  • Mejora la experiencia sin limitar la entrada.
<input type="text" list="ciudades" name="ciudad">

<datalist id="ciudades">
    <option value="Quito">
    <option value="Guayaquil">
    <option value="Cuenca">
</datalist>

Diferencias clave entre los elementos seleccionables

  • radio: una sola opción obligatoria.
  • checkbox: múltiples opciones independientes.
  • select: lista cerrada de opciones.
  • datalist: sugerencias abiertas para el usuario.
