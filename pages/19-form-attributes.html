ATRIBUTOS DE LOS FORMULARIOS EN HTML

Los atributos permiten controlar el comportamiento, la validación y el funcionamiento de los elementos dentro de un formulario. Estos atributos pueden aplicarse a diferentes etiquetas como <input>, <textarea>, <select>, entre otras.

1. Atributos Para Restricciones De Contenido

maxlength

Define la cantidad máxima de caracteres permitidos.

minlength

Establece el mínimo de caracteres que el usuario debe ingresar.

min y max

Controlan los valores mínimos y máximos permitidos en campos numéricos o de fecha.

pattern

Permite validar el contenido mediante una expresión regular.

2. Atributos De Ayuda Y Comportamiento

placeholder

Muestra un texto guía dentro del campo antes de que el usuario escriba.

required

Indica que el campo es obligatorio para enviar el formulario.

readonly

Hace que el campo no pueda ser modificado, pero su valor sí será enviado.

disabled

Deshabilita el campo por completo: no se puede editar, no recibe foco y no se envía.

autofocus

Hace que el campo obtenga el foco automáticamente al cargar la página.

3. Atributos Relacionados Con La Selección

selected

Marca una opción como seleccionada por defecto dentro de un elemento <select>.

checked

Marca por defecto una casilla de tipo checkbox o un botón radio.

4. Atributos Esenciales Para Envío De Datos

name

Es el nombre del campo. Si no está presente, su valor no se enviará al servidor.
