VIDEOS EN HTML

HTML permite insertar y reproducir contenido multimedia nativo sin depender de plugins externos. La etiqueta principal para trabajar con videos es <video>.

1. Etiqueta <video>

Es la etiqueta principal para mostrar videos en una página web.

Ejemplo Básico:

<video src="video.mp4" controls></video>

Atributos Principales:

  • controls: muestra los controles del reproductor (play, pausa, volumen, etc.).
  • autoplay: reproduce el video automáticamente al cargar la página.
  • loop: reproduce el video en bucle.
  • muted: inicia el video sin sonido.
  • poster: muestra una imagen previa antes de reproducir el video.
  • width y height: definen el tamaño del reproductor.
  • preload: controla si el video debe cargarse antes de reproducción (none, metadata o auto).

2. Etiqueta <source>

Permite definir múltiples archivos de video con distintos formatos para mejorar compatibilidad.

Ejemplo:

<video controls>
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    Tu navegador no soporta el video.
</video>

3. Formatos De Video Compatibles

  • MP4 (H.264 + AAC): el formato más compatible entre navegadores.
  • WebM: buena compresión y calidad. Formato abierto.
  • Ogg/Theora: menos usado, pero soportado por navegadores modernos.

Formato Recomendado:

El formato más aconsejable es MP4 por su compatibilidad universal en prácticamente todos los navegadores y dispositivos.

4. Subtítulos Y Pistas (<track>)

Se usa dentro de <video> para agregar subtítulos, descripciones de audio u otros tipos de pistas.

Ejemplo:

<video controls>
    <source src="video.mp4" type="video/mp4">
    <track src="subtitulos.vtt" kind="subtitles" srclang="es" label="Español">
</video>

5. Buenas Prácticas

  • Optimizar el tamaño del archivo para mejorar el rendimiento.
  • Incluir varios formatos usando <source>.
  • Usar controls para mejorar usabilidad.
  • Evitar autoplay con sonido porque afecta la experiencia del usuario.
  • Utilizar poster para mostrar una miniatura atractiva.
