IFRAMES EN HTML

La etiqueta <iframe> permite incrustar contenido externo dentro de una página web. Puede incluir otras páginas HTML, videos, mapas, documentos y más.

1. ¿Qué Es Un Iframe?

Un iframe (Inline Frame) es un marco embebido que carga una página o recurso externo dentro del documento actual.

Ejemplo Básico:

<iframe src="pagina.html"></iframe>

2. Atributos Principales De <iframe>

  • src: URL o ruta del contenido a mostrar.
  • width: ancho del iframe.
  • height: alto del iframe.
  • title: describe el contenido para accesibilidad.
  • loading: controla la carga diferida (lazy).
  • allow: permite permisos al contenido incrustado.
  • sandbox: restringe funcionalidades por seguridad.
  • referrerpolicy: controla si se envía información del referidor.

Ejemplo Con Atributos:

<iframe 
    src="https://example.com"
    width="600"
    height="400"
    title="Contenido externo"
    loading="lazy">
</iframe>

3. Seguridad En Iframes

Los iframes pueden representar riesgos de seguridad si cargan contenido no confiable. Para mitigarlos, existen atributos específicos:

3.1 Atributo sandbox

Restringe lo que puede hacer el contenido del iframe. Sin parámetros, bloquea casi todo.

  • allow-scripts: permite ejecutar scripts.
  • allow-same-origin: permite tratar el contenido como del mismo origen.
  • allow-forms: permite enviar formularios.
  • allow-popups: permite abrir ventanas emergentes.
<iframe src="pagina.html" sandbox="allow-scripts allow-forms"></iframe>

3.2 Atributo allow

Define qué permisos se otorgan al iframe.

  • camera: acceso a la cámara.
  • microphone: acceso al micrófono.
  • fullscreen: permitir modo pantalla completa.
<iframe src="video.html" allow="fullscreen"></iframe>

4. Casos Comunes De Uso

  • Incrustar mapas (Google Maps).
  • Incrustar videos desde plataformas externas.
  • Mostrar contenido HTML independiente.
  • Mostrar widgets externos (clima, feeds, etc.).
  • Incrustar documentos (PDF, presentaciones, etc.).

Ejemplo De Uso Con Google Maps:

<iframe 
    src="https://www.google.com/maps/embed?pb=!1m18..."
    width="600"
    height="450"
    style="border:0;"
    allowfullscreen=""
    loading="lazy">
</iframe>

5. Buenas Prácticas

  • Usar title para accesibilidad.
  • Preferir loading="lazy" para mejorar rendimiento.
  • Aplicar sandbox al contenido no confiable.
  • Evitar cargar muchos iframes en una sola página.
  • Utilizar dimensiones proporcionales (CSS) para videos e incrustaciones.
