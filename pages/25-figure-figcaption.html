FIGURE Y FIGCAPTION EN HTML

Las etiquetas <figure> y <figcaption> se utilizan para agrupar contenido visual o ilustrativo junto con una descripción o leyenda relacionada. Estas etiquetas mejoran la semántica y la accesibilidad del documento.

1. ¿Qué Es La Etiqueta <figure>?

<figure> es un contenedor semántico diseñado para agrupar contenido independiente del flujo principal, generalmente medios como imágenes, gráficos, código, tablas u otros elementos ilustrativos.

  • Representa un contenido que podría moverse de lugar sin afectar el significado del texto.
  • Mejora la semántica del documento.
  • Se usa comúnmente para imágenes con descripción.

Ejemplo Básico:

<figure>
    <img src="imagen.jpg" alt="Descripción de imagen">
</figure>

2. ¿Qué Es La Etiqueta <figcaption>?

<figcaption> proporciona una leyenda o descripción para el contenido dentro de <figure>. Es opcional pero muy útil para accesibilidad y claridad.

  • Debe colocarse como hijo directo de <figure>.
  • Puede ubicarse antes o después del contenido ilustrativo.
  • Se utiliza para explicar o contextualizar la imagen o figura.

Ejemplo Con Figcaption:

<figure>
    <img src="playa.jpg" alt="Playa al atardecer">
    <figcaption>Fotografía de una playa al atardecer.</figcaption>
</figure>

3. Atributos De <figure> Y <figcaption>

Las etiquetas <figure> y <figcaption> no poseen atributos propios especiales, pero admiten los atributos globales de HTML:

  • id: identifica la figura.
  • class: permite aplicar estilos.
  • style: define estilos en línea.
  • title: añade información contextual.

Ejemplo Con Atributos Globales:

<figure id="grafico-ventas" class="figura-destacada">
    <img src="grafico.png" alt="Gráfico de ventas 2025">
    <figcaption>Gráfico oficial de ventas del año 2025.</figcaption>
</figure>

4. ¿Cuándo Usar <figure>?

Es recomendable utilizar esta etiqueta en los siguientes casos:

  • Cuando la imagen requiere una explicación adicional.
  • Cuando el contenido es independiente del texto.
  • Cuando se necesita un elemento más semántico para SEO y accesibilidad.
  • Para gráficos, code snippets, diagramas, tablas ilustrativas.

Ejemplo Con Código Dentro De Figure:

<figure>
    <pre><code>console.log("Hola Mundo");</code></pre>
    <figcaption>Ejemplo de impresión en consola con JavaScript.</figcaption>
</figure>

5. Beneficios Del Uso De Figure Y Figcaption

  • Mejor semántica del documento HTML.
  • Mayor accesibilidad para personas que usan lectores de pantalla.
  • Contenido visual más organizado.
  • Posicionamiento SEO mejorado.
  • Separación clara entre contenido principal y contenido complementario.
