ETIQUETAS META EN HTML

Las etiquetas <meta> se utilizan dentro de la sección <head> para definir metadatos sobre el documento. Estos datos no son visibles en la página, pero sí son esenciales para navegadores, motores de búsqueda y aplicaciones internas del sistema.

1. ¿Qué Es Una tiqueta Meta?

Una etiqueta <meta> es un elemento vacío (no tiene etiqueta de cierre) que proporciona información extra sobre la página. Los navegadores la usan para interpretar correctamente el documento y ciertos servicios externos pueden utilizarla para extraer información técnica.

  • No se muestra en pantalla.
  • Debe colocarse siempre dentro del <head>.
  • Ayuda a controlar comportamientos del navegador.

Ejemplo Básico:

<meta name="description" content="Página sobre etiquetas meta en HTML">

2. Atributos Principales De La Etiqueta <meta>

  • name: especifica el tipo de metadato (autor, descripción, etc.).
  • content: contiene la información del metadato.
  • charset: establece la codificación del documento.
  • http-equiv: permite definir encabezados HTTP dentro del HTML.

Ejemplos:

<meta charset="UTF-8">
<meta name="author" content="Juan Pérez">
<meta http-equiv="refresh" content="20">

3. Tipos Comunes De Metaetiquetas

3.1. Codificación De Caracteres

Define cómo se interpretan los caracteres especiales del documento.

<meta charset="UTF-8">

3.2. Meta Viewport (importante para móviles)

Ajusta el ancho y la escala del contenido para que se vea correctamente en dispositivos móviles.

<meta name="viewport" content="width=device-width, initial-scale=1.0">

3.3. Descripción Del Sitio

Proporciona un resumen breve de la página, útil para buscadores.

<meta name="description" content="Curso completo de HTML">

3.4. Palabras Clave (obsoleto para SEO moderno)

Pueden usarse, pero los motores de búsqueda ya no consideran este metadato para posicionamiento.

<meta name="keywords" content="HTML, programación, desarrollo web">

3.5. Autor Del Documento

Especifica quién creó la página.

<meta name="author" content="Nombre del autor">

3.6. Control De Caché

Utiliza http-equiv para definir directrices de almacenamiento.

<meta http-equiv="cache-control" content="no-cache">

3.7. Redirecciones Automáticas

Permiten recargar la página o enviar al usuario a otra ruta después de un tiempo.

<meta http-equiv="refresh" content="5; url=otra-pagina.html">

4. Importancia DDl Uso De Metaetiquetas

  • Mejoran la accesibilidad del documento.
  • Ayudan a los motores de búsqueda a interpretar el contenido.
  • Permiten controlar cómo se visualiza la página en dispositivos móviles.
  • Definen parámetros técnicos como codificación o comportamiento del navegador.
