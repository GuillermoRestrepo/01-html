PROTOCOLO OPEN GRAPH

El protocolo Open Graph (OGP) fue creado por Facebook y permite controlar cómo se muestra una página cuando se comparte en redes sociales como Facebook, WhatsApp, Telegram, LinkedIn y más.

1. ¿Para Qué Sirve?

  • Define la imagen que se mostrará al compartir el enlace.
  • Permite establecer título y descripción personalizados.
  • Mejora la apariencia del enlace en redes sociales.
  • Da contexto semántico a la URL compartida.

2. Etiquetas Principales De Open Graph

  • og:title: Título del contenido.
  • og:description: Descripción breve.
  • og:image: Imagen a mostrar.
  • og:url: URL canónica del sitio.
  • og:type: Tipo de objeto (website, article, etc.).
  • og:site_name: Nombre del sitio.

Ejemplo:

<meta property="og:title" content="Mi Página">
<meta property="og:description" content="Descripción breve.">
<meta property="og:image" content="https://ejemplo.com/imagen.jpg">
<meta property="og:url" content="https://ejemplo.com">
<meta property="og:type" content="website">

3. Buenas Prácticas

  • La imagen debe medir mínimo 1200×630 px.
  • Usar formatos JPG o WEBP.
  • Incluir el mayor número posible de etiquetas OG.
  • Preferir URL absolutas.
