ICONOS Y FAVICONS EN HTML

Los íconos y favicons se utilizan para identificar un sitio web en navegadores, pestañas, marcadores, accesos directos y dispositivos móviles. Se definen mediante la etiqueta <link> dentro del <head>.

1. ¿Qué Es Un Favicon?

  • Es el pequeño ícono que aparece en la pestaña del navegador.
  • Representa visualmente al sitio web.
  • Puede mostrarse también en marcadores o historial.

2. Formatos Recomendados

  • .ico — El más compatible con todos los navegadores.
  • .png — Alta calidad, recomendado para pantallas modernas.
  • .svg — Escalable, ideal para diseños limpios (no compatible con todos los navegadores antiguos).

3. Tamaños Comunes

  • 16×16 px — Pestañas del navegador.
  • 32×32 px — Favoritos o accesos directos.
  • 180×180 px — Icono para Apple Touch.
  • 192×192 / 512×512 px — Para Android o PWA.

4. Cómo Agregar Un Favicon

Se debe añadir dentro de <head> usando la etiqueta <link>.

Ejemplo básico:

<link rel="icon" href="favicon.ico" type="image/x-icon">

Ejemplo usando PNG:

<link rel="icon" href="favicon.png" type="image/png">

5. Iconos Para Dispositivos Móviles

Los dispositivos Apple requieren un ícono especial.

<link rel="apple-touch-icon" href="apple-icon.png">

Para Android o PWA:

<link rel="icon" sizes="192x192" href="icon-192.png">
<link rel="icon" sizes="512x512" href="icon-512.png">

6. Buenas pPrácticas

  • Incluir varios tamaños para mejor compatibilidad.
  • Usar imágenes simples y de buena resolución.
  • Preferir PNG para mejor calidad, ICO para compatibilidad total.
  • Utilizar rutas absolutas si el ícono no carga correctamente.
